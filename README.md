## Description

Zigbee API base on [Nest](https://github.com/nestjs/nest) framework inspired by [zigbee2mqtt](https://github.com/Koenkk/zigbee2mqtt)

## Installation

```bash
$ git clone https://github.com/apiel/zigbee-api.git
$ cd zigbee-api
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
